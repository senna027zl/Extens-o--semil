# Instruções para o Agente Customizado: Mnemosyne Extension

## Visão Geral
A extensão Mnemosyne é um sistema cognitivo de memória para o personagem Hanna em roleplays. Ela analisa blocos de mensagens do chat, gera resumos densos usando IA (via NanoGPT API), e salva esses resumos como Gists privados no GitHub. Além disso, injeta memória acumulada no prompt do roleplay para manter consistência comportamental.

## Funcionalidades Principais

### 1. Configuração e Armazenamento
- **API Key**: Chave de autenticação para a API NanoGPT (nano-gpt.com). Necessária para gerar resumos.
- **GitHub Token**: Token pessoal do GitHub com permissões para criar Gists. Usado para salvar resumos como Gists privados.
- **Modelo de IA**: Modelo usado na API (padrão: deepseek-v3.2).
- **Intervalo de Mensagens**: Número de mensagens por bloco (padrão: 50). Define quando um novo bloco é processado.
- **Prompt de Resumo**: Texto editável que instrui a IA sobre como analisar e resumir os blocos. Focado em arco principal, momentos-chave, estado emocional de Hanna, padrões comportamentais e sinais somáticos.
- **Módulo Ativo**: Ativa/desativa o processamento automático.
- **Injeção no RP**: Quando ativado, lê os Gists salvos e injeta um prompt de memória no roleplay, influenciando o comportamento de Hanna baseado no histórico.

### 2. Processamento de Blocos
- **Automático**: A cada nova mensagem recebida, verifica se o número de mensagens desde o último processamento atingiu o intervalo. Se sim, processa o bloco atual automaticamente.
- **Manual**: Botão "↺ Resumir agora" força o processamento do bloco atual.
- **Bloco Específico**: Permite resumir um bloco passado pelo número (ex: bloco 5).
- **Salvar Tudo**: Processa e salva todos os blocos do chat atual como Gists de uma vez.

### 3. Salvamento e Recuperação
- **Criação de Gists**: Cada resumo é salvo como um Gist privado no GitHub, com descrição formatada (ex: "Mnemosyne — Bloco 1 — 26/04/2026") e conteúdo em Markdown.
- **Listagem de Gists**: Exibe os últimos 5 Gists na interface, com preview do conteúdo, hora de criação e tags extraídas (vulnerabilidade, insubordinação, afeto, controle, teste).
- **Contador de Gists**: Mostra o total de Gists salvos.

### 4. Injeção de Estado/Memória
- Analisa os últimos 10 resumos para extrair estado geral (vulnerabilidade, teste de limites, contenção).
- Injeta um prompt no roleplay (ex: "[Memória acumulada: histórico de vulnerabilidade — instinto de proteção ativo.]") para guiar o comportamento de Hanna.

### 5. Interface de Usuário
- Integrada nas configurações de extensões do aplicativo.
- Campos para todas as configurações.
- Botões para ações manuais.
- Área de status para feedback (pronto, processando, erros).
- Lista rolável dos últimos Gists.

## Comandos e Ações Disponíveis
- **Salvar Configurações**: Atualiza e salva todas as configurações no localStorage.
- **Resumir Agora**: Processa o bloco atual manualmente.
- **Resumir Bloco Específico**: Processa um bloco passado pelo número inserido.
- **Salvar Tudo**: Processa todos os blocos do chat e salva como Gists.
- **Alternar Injeção no RP**: Liga/desliga a injeção de memória no roleplay, atualizando o prompt imediatamente.

## Eventos Automáticos
- **APP_READY**: Inicializa sincronização e carrega lista de Gists.
- **CHAT_CHANGED**: Recarrega lista de Gists.
- **MESSAGE_RECEIVED**: Verifica e processa bloco se intervalo atingido.

## Dependências e APIs
- **NanoGPT API**: Para geração de resumos (https://nano-gpt.com/api/v1/chat/completions).
- **GitHub API**: Para criação e listagem de Gists (https://api.github.com/gists).

## Notas Técnicas
- Resumos são gerados em português, focados na dinâmica Hanna-Senna.
- Gists são privados e nomeados sequencialmente (bloco-001.md, etc.).
- Estado é extraído via regex e análise de texto dos resumos.
- Configurações são persistidas no localStorage com chave 'mnemosyne-settings'.
- Logs no console para debugging (ex: "[Mnemosyne] Gist criado: URL").

Esta extensão permite uma memória persistente e evolutiva para o personagem, enriquecendo o roleplay com continuidade emocional e comportamental.
